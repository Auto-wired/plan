# 일정 추가 기능 명세

> 공통: [EVENT_COMMON.md](./EVENT_COMMON.md) · [TOAST.md](./TOAST.md)  
> 인덱스: [README.md](./README.md)

---

## 1. 진입

달력 **빈 공간**을 클릭(또는 드래그 선택)하여 **일정 추가 Modal**을 띄운다.

---

## 2. 입력 항목

| 항목 | 필수 여부 |
|------|-----------|
| 제목 | 필수 |
| 설명 | 선택 |
| 종일 여부 | 필수 |
| 시작일 | 필수 |
| 종료일 | 필수 |
| 반복 옵션 | 필수 |
| 카테고리 | 필수 |

---

## 3. 검증 및 실패

[EVENT_COMMON.md](./EVENT_COMMON.md)의 `validateEventForm`을 따른다.

필수 항목 미입력 시 토스트:

- variant: `error`
- title: `일정 추가 실패`
- description: `{실패 사유}`

---

## 4. 종일 여부

- **기본값:** 활성화(종일)
- **종일 활성화:** 시작일·종료일에 **시간 입력 UI 없음**. DB 저장 시 해당 날짜 `00:00:00`.
- **종일 비활성화:** 시작일·종료일에 **시간 지정 가능**.

---

## 5. 반복 옵션

- 선택지: **반복 안 함**, **매일**, **매주**, **매월**, **매년**
- **기본값:** 반복 안 함

반복 안 함을 제외하면 **반복 세부 옵션** UI가 표시된다.

- **반복 간격:** N일/주/월/년마다
- **반복 종료**

### 반복 종료 옵션

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| 계속 반복 | 종료 조건 없음 | **기본** |
| 횟수 | 지정 횟수만큼 | 횟수 입력 UI |
| 종료 날짜 | 지정 날짜까지 | 날짜 입력 UI |
